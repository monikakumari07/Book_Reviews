Project 

Description:
This project is a web application that allows users to register, log in, and
write reviews for books. Users can search for books, view details about authors and books, and read reviews by other users.

Features:
- User Registration and Authentication: Users can register and log in using JWT authentication.
- Author Management: Manage authors with CRUD operations.
- Book Management: Manage books with CRUD operations, including filtering by author and publication date.
- Review Management: Users can write, edit, and delete reviews for books.
- Nested Serialization: Includes related data in API responses.

Installation:
1. Clone the repository to your local machine.
2. Install the required dependencies using pip.
3. Set up the Django Rest Framework project and database.
4. Run the Django Rest Framework development server.
